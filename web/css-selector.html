CSS セレクターのいろいろ、初心者の私が良くわからなくなり、使いながら調べたが、一覧を整理しておく

基本セレクター

すべての要素：*

すべての要素を選択する

* {
  color: green;
}

HTMLタグ／クラス名／ID

HTML

<p class="class_name">
  <div> title </div>
  <span id="id_name">A green span</span> in a red paragraph.
</p>

HTMLタグ名を使う

div {
  color: green;
}

クラス名を使う

.class_name {
  background-color: gray;
}

IDを使う

.id_name {
  color: red;
}

属性

構文 説明
[attr] attr という名前の属性を持つ要素を表します。
[attr=value] attr という名前の属性の値が正確に value である要素を表します。
[attr^=value] attr という名前の属性の値が value で始まる要素を表します。
[attr$=value] attr という名前の属性の値が value で終わる要素を表します。
[attr*=value] attr という名前の属性の値が、文字列中に value を1つ以上含む要素を表します。
[attr~=value] attr という名前の属性の値が正確に value と一致する要素を表します。空白区切りの語のリストの形で、複数の value を含めることができます。attr="value other"も合致する
[attr|=value] attr という名前の属性の値が正確に value と一致するか、 value で始まり直後にハイフン (-) が続く要素を表します。attr="value"/attr="value-1"/attr="value-2"が合致する
[attr operator value i] 閉じ角括弧の前に i (または I) を追加すると、 (ASCII の範囲内の文字の場合) 値は大文字と小文字を区別せずに比較されます。

セレクターの結合

AND/OR/NOT演算

プログラムをよく書く人は、CSSもif分のように指定したくなると思う。CSSはif分がないが、AND/OR/NOT関係指定できる。

構文 説明
AB AND演算、AとB両方の条件が成立の要素を選択。#title.dark：id="title" かつ class="dark"の要素を選択
A, B OR演算、AかBの要素を選択。#title .dark：id="title" か class="dark"の要素を選択
:not(A) NOT演算、Aではない要素を選択。:not(AB)や:not(A,B)でも指定可能

子孫関係など

構文　　　 説明
A B 子孫選択、第1の要素の子孫にあたるノードを選択します。div span は <div> 要素の中にある <span> 要素をすべて選択します。
A > B 子選択、第1の要素の直接の子に当たるノードを選択します。ul > li は <ul> 要素の内側に直接ネストされた <li> 要素をすべて選択します。
A ~ B 兄弟選択、第2の要素が第1の要素の後にあり（ただし直後でなくても構わない）、両者が同じ親を持つ場合です。p ~ span は <p> 要素の後にある <span> 要素をすべて選択します。
A + B 隣接兄弟選択、第2の要素が第1の要素の直後にあり、両者が同じ親を持つ場合です。h2 + p は <h2> 要素の後にすぐに続く <p> 要素をすべて選択します。
