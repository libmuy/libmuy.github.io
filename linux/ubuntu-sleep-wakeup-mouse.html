Ubuntu20.04：スリップから復帰時、マウスが使えない時の対処

25 Jun 2020 06 Oct 2021

Contents
  • 対処方法

スリープから復帰時、マウスが動作しなかった。
原因は、マウスのドライバがスリープ対応していないようだ。

対処方法

  1. psmouseを使っているかを確認 
    lsmod | grep psmouse
  2. スクリプトを作成 
    $ cd /lib/systemd/system-sleep/
$ sudo vi fixtouchpad
  3. スクリプト内容 
    #!/bin/bash
[ "$1" = "post" ] && exec modprobe psmouse
[ "$1" = "pre" ] && exec rmmod psmouse
exit 0
  4. 実行権限を付与 
    sudo chmod +x fixtouchpad
