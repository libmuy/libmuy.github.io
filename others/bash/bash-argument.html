Bashの関数やスクリプト自体の引数について

引数一覧

引数 説明
$# 実行時に指定された引数の数を表す変数。
$@ / $* シェルスクリプト実行時、もしくは set コマンド実行時に指定された全パラメータが設定される変数。
$0 実行時のコマンド名が設定される変数。c言語のargv[0]と似ている
$1 - $n シェルスクリプト実行時に指定した引数の値がそれぞれ設定される変数。

変数 $@ と $* が基本的に同じだが、"" で囲んだときの動作が異なる。

$@と$*の違い

  • test.sh 引数の数と引数を出力 
      echo '$#  :' \"$#\"
  cnt=$#

  for (( i = 0; i < $cnt; i++ )); do
          echo "arg[$i]: $1"
          shift
  done
  • 結果 
      ~/wk$ set a b c
  ~/wk$ /tmp/test.sh "$@"
  $#  : "3"
  arg[0]: a
  arg[1]: b
  arg[2]: c
  ~/wk$ /tmp/test.sh "$*"
  $#  : "1"
  arg[0]: a b c
