Ubuntuで、エアキャプチャする方法

25 Jun 2020

Contents

いつもMacでエアキャプチャしていますが、Macがないときに、Ubuntuでも使えますよ。

  1. ネットワークマネージャーを無効 
    sudo systemctl stop NetworkManager
  2. ネットワークインターフェースを作成 
    sudo iw phy phy0 interface add wlanmon0 type monitor
sudo ifconfig wlanmon0 up
  3. チャンネルを設定 
    sudo iwconfig wlanmon0 channel 11
